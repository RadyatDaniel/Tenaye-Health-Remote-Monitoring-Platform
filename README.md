# Tenaye Health Remote Monitoring Platform

Tenaye Health is a remote patient monitoring platform designed to bridge the gap between healthcare providers and patients. The platform enables real-time health tracking, data visualization, and seamless communication to improve healthcare delivery.

## 🛠️ Tech Stack & Configuration

The frontend application is built using a modern, ultra-fast web development stack:

* **Framework:** [React](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/) (with Fast Hot Module Replacement - HMR)
* **Linter:** [ESLint](https://eslint.org/) for clean, standardized code architecture.

### Plugins Used
* `@vitejs/plugin-react` uses [Oxc](https://oxc-project.github.io/) for fast compilation performance.
* *Alternative option available:* `@vitejs/plugin-react-swc` using SWC.

> ⚠️ **Note on React Compiler:** The React Compiler is explicitly disabled in this configuration template to maintain optimal local development and production build speeds.
